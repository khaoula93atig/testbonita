Exported formats

The .zip archive of an exported page contains:

  • JSON descriptor to exchange between UI Designers
  • Minified widgets JavaScript for optimized performance
  • Standard widgets JavaScript for edition outside the UI Designer

Attention: if you edit the page, form or layout outside the UI Designer, you will not be able to reimport it. You will have to manage the minification and update the resources URL to trigger the cache busting. For more information, consult the documentation.
